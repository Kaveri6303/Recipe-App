# Recipe-App  
A responsive recipe search application built with HTML, CSS, and JavaScript.  

## Features  
- Real-time recipe search  
- API integration using Fetch  
- Responsive UI with cards and detail views  

## Tech Stack  
- HTML, CSS, JavaScript (ES6+)  
- Fetch API
